#### 查找某一个系统调用函数的头文件
- man -f func
- man num func

```
man -f mkdir
```
出现：
```
mkdir                (1)  - make directories
mkdir                (1p)  - make directories
mkdir                (2)  - create a directory
mkdir                (3p)  - make a directory
```

```
man -1 mkdir
man -1p mkdir
man -2 mkdir
man -3p mkdir
```
